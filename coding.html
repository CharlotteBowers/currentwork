Links I find useful

Some useful links for learning code:

Codecademy
If you’re new to coding and need to hear it being taught in your own language and not alien then Codecademy is the place for you! It makes the basics of coding look easy and has forums just incase you get stuck. Although they don’t have every coding language on there yet, they’re looking to expand their courses and the courses currently available are good quality.

Free Code Camp:
Unlike Codecademy, this goes into more detail. Where Codecademy gives you the basics and a place to carry out your practise, Free Code Camp tells you how to set up your own projects and deploy them. Not to mention you get a certificate that’s actually worth something if you put in the hours!

Stack Overflow
A must for developers of all. This is a platform to go for help or to help other people with any kind of coding problem.

jsfiddle
This is a brilliant site to go if you need a space to mess about or test HTML/CSS/JS and don’t want to be opening a million programmes, it makes it easy to edit and see the result. Goes hand in hand with learning from Codecademy.
